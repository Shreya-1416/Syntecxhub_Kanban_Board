# Kanban Board

A modern, pastel-themed Kanban Board built using **HTML, CSS, and JavaScript**.

This project allows users to manage tasks visually using drag-and-drop functionality with priority levels and local storage persistence.

---

## Features

-  Four Columns: Backlog, In Progress, Review, Complete
-  Drag & Drop between columns
-  Priority badges (Low / Medium / High)
-  Double-click to edit tasks
-  Delete task button
-  Live search filter
-  Confetti animation when task moves to Complete
-  Cute pastel Dark Mode (saved in localStorage)
-  Data persistence using localStorage
-  Responsive design

---

## Technologies Used

- HTML5
- CSS3 (Flexbox + Gradients + Soft UI Design)
- JavaScript (DOM Manipulation, Drag & Drop API)
- localStorage API
- Canvas Confetti Library
